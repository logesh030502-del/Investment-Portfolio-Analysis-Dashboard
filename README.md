** Investment Portfolio Analysis & Dashboard (Power BI)**
** Project Overview**

This project focuses on analyzing a diversified investment portfolio using Power BI. The dashboard provides a comprehensive view of different asset classes including Gold, Provident Fund (PF), Mutual Funds, Government Bonds, and Fixed Deposits (FDs).

The objective is to track performance, understand asset allocation, and evaluate risk vs return to support better financial decision-making.

** Objectives**

Monitor overall portfolio performance
Compare returns across different investment types
Analyze asset allocation and diversification
Evaluate risk and stability of investments
Provide insights for optimizing investment strategy

**Investment Categories Covered**

 Gold – Safe-haven asset, tracked for value appreciation
 Provident Fund (PF) – Long-term retirement savings with stable returns
 Mutual Funds – Market-linked investments with growth potential
 Government Bonds – Low-risk fixed income securities
 Fixed Deposits (FDs) – Secure investments with fixed returns
 
**** Key Metrics & KPIs****

Total Portfolio Value
Return on Investment (ROI)
Portfolio Growth Over Time
Asset Allocation (%)
Individual Asset Performance
Risk vs Return Comparison

**Tools & Technologies**

Power BI – Data visualization & dashboard creation
DAX – Calculated measures and KPIs
Excel – Data source and preprocessing

**Dashboard Features**
Interactive filters and slicers for asset selection
Time-based trend analysis of portfolio growth
Visual comparison of different investment categories
Dynamic KPI cards for quick insights
Clean and user-friendly layout for better understanding
 
**Insights Derived**
Identified which asset class provides the highest returns
Evaluated stability of low-risk investments like PF, Bonds, and FDs
Compared market-driven vs fixed-return investments
Helped in understanding diversification effectiveness

 **Conclusion**

This dashboard enables a clear understanding of how different investments perform within a portfolio. It helps investors make informed decisions by balancing risk, return, and diversification.
